\doxysubsection{Descripción detallada}
The {\ttfamily \mbox{\hyperlink{classedu_1_1duke_1_1_image_resource}{Image\+Resource}}} class represents an image as a grid of {\ttfamily \mbox{\hyperlink{classedu_1_1duke_1_1_pixel}{Pixel}}} objects and allows access to all of them, using the method {\ttfamily pixels}. These pixels can then be iterated over using a {\ttfamily for} loop.

Note, no changes made to the pixels in this image affect the original image opened unless you use the method {\ttfamily save}.

Example usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classedu_1_1duke_1_1_image_resource}{ImageResource}} image = new \mbox{\hyperlink{classedu_1_1duke_1_1_image_resource_a33af371a101e8d364ab27cc63832c7e2}{ImageResource()}};
for (\mbox{\hyperlink{classedu_1_1duke_1_1_pixel}{Pixel}} p : image.pixels()) \{
    int red = p.getRed();
    int green = p.getGreen();
    int blue = p.getBlue();
    int average = (red + green + blue) / 3;
    p.setRed(average);
    p.setGreen(average);
    p.setBlue(average);
\}
image.draw();
\end{DoxyPre}
