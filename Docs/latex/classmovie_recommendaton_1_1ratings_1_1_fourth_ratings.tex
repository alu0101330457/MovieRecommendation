Computes a similarity rating for each rater in the Rater\+Database (except the rater with the ID given by the parameter) to see how similar they are to the Rater whose ID is the parameter to get\+Similarities. 

Note that in each \mbox{\hyperlink{classmovie_recommendaton_1_1ratings_1_1_rating}{Rating}} object the item field is a rater’s ID, and the value field is the dot product comparison between that rater and the rater whose ID is the parameter to get\+Similarities. 
\begin{DoxyParams}{Parámetros}
{\em id} & \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Devuelve}
Array\+List$<$\+Rating$>$ sorted by ratings from highest to lowest rating with the highest rating first and only including those raters who have a positive similarity rating since those with negative values are not similar in any way.
