\doxysubsection{Descripción detallada}
The {\ttfamily \mbox{\hyperlink{classedu_1_1duke_1_1_storage_resource}{Storage\+Resource}}} class stores any number of {\ttfamily String} objects and allows access to these stored values one at a time, using the method {\ttfamily data}. These strings can then be iterated over in the order they were added using a {\ttfamily for} loop.

This class mirrors an {\ttfamily Array\+List\texorpdfstring{$<$}{<}String\texorpdfstring{$>$}{>}} in some functionality, but is simpler to use and fits the Duke/\+Coursersa model of creating and using iterables.

Example usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classedu_1_1duke_1_1_file_resource}{FileResource}} fr = new FileResource();
\mbox{\hyperlink{classedu_1_1duke_1_1_storage_resource}{StorageResource}} store = new \mbox{\hyperlink{classedu_1_1duke_1_1_storage_resource_a06da2137a8e9d4a6597d3ac5f9795577}{StorageResource()}};
for (String s : fr.words()) \{
    store.add(s);
\}
// can process store here, e.g.,
// get number of strings stored
int x = store.size();
for (String s : store.data()) \{
    // print or process s
\}
\end{DoxyPre}
