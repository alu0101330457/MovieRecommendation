6   /**
7    * Implement this interface to allow your code to be integrated with our
8    * web site.
9    * 
10   * When users first visit the recommender website, our code will call the
11   * method <code>getItemsToRate()</code> to get a list of movies to display
12   * on the web page for users to rate.
13   * 
14   * When a user submits their ratings, our code will call the method <code>
15   * printRecommendationsFor</code> to get your recommendations based on the
16   * user's ratings.  The ID given to this method is for a new Rater that we 
17   * have already added to the RaterDatabase with ratings for the movies 
18   * returned by the first method.  Whatever is printed from that method will 
19   * be displayed on the web page: HTML, plain text, or debugging information.
20   * 
21   */
22  public interface Recommender {
23      /**
24       * This method returns a list of movie IDs that will be used to look up 
25       * the movies in the MovieDatabase and present them to users to rate. 
26       *  
27       * The movies returned in the list will be displayed on a web page, so
28       * the number you choose may affect how long the page takes to load and
29       * how willing users are to rate the movies.  For example, 10-20 should
30       * be fine, 50 or more would be too many.
31       * 
32       * There are no restrictions on the method you use to generate this list
33       * of movies: the most recent movies, movies from a specific genre, 
34       * randomly chosen movies, or simply your favorite movies.
35       * 
36       * The ratings for these movies will make the profile for a new Rater 
37       * that will be used to compare to for finding recommendations.
38       */
39      public ArrayList<String> getItemsToRate ();
40  
41      /**
42       * This method returns nothing, but prints out an HTML table of the 
43       * movies recommended for the given rater.
44       * 
45       * The HTML printed will be displayed on a web page, so the number you
46       * choose to display may affect how long the page takes to load.  For 
47       * example, you may want to limit the number printed to only the top 
48       * 20-50 movies recommended or to movies not rater by the given rater.
49       * 
50       * You may also include CSS styling for your table using the &lt;style&gt;
51       * tag before you print the table.  There are no restrictions on which 
52       * movies you print, what order you print them in, or what information
53       * you include about each movie. 
54       * 
55       * @param webRaterID the ID of a new Rater that has been already added to 
56       *        the RaterDatabase with ratings for the movies returned by the 
57       *        method getItemsToRate
58       */
59      public void printRecommendationsFor (String webRaterID);
60  }
