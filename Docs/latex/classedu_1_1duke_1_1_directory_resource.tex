\doxysubsection{Descripción detallada}
The {\ttfamily \mbox{\hyperlink{classedu_1_1duke_1_1_directory_resource}{Directory\+Resource}}} class allows the user to choose one or more files from a directory (or folder) with a file selection dialog box by using the method {\ttfamily selected\+Files}. These files can then be iterated over using a {\ttfamily for} loop.

Example usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classedu_1_1duke_1_1_directory_resource}{DirectoryResource}} dr = new \mbox{\hyperlink{classedu_1_1duke_1_1_directory_resource_a06968fba0d0b6682c8cb2b6731c9022f}{DirectoryResource()}};
for (File f : dr.selectedFiles()) \{
    \mbox{\hyperlink{classedu_1_1duke_1_1_image_resource}{ImageResource}} ir = new ImageResource(f);
    ir.draw();
\}
\end{DoxyPre}
