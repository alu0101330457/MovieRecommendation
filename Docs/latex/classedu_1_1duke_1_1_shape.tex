\doxysubsection{Descripción detallada}
The {\ttfamily \mbox{\hyperlink{classedu_1_1duke_1_1_shape}{Shape}}} class represents a collection of {\ttfamily \mbox{\hyperlink{classedu_1_1duke_1_1_point}{Point}}}s that can easily model a polygonal shape. Points can be added in order via code or given using a file, one point per line separated by a comma. The points can then be accessed in the order they were added one at a time, using the method {\ttfamily get\+Points}.

Example usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classedu_1_1duke_1_1_point}{Point}} a = new Point(3, 4);
\mbox{\hyperlink{classedu_1_1duke_1_1_point}{Point}} b = new Point(2, 5);
\mbox{\hyperlink{classedu_1_1duke_1_1_shape}{Shape}} s = new \mbox{\hyperlink{classedu_1_1duke_1_1_shape_a3ca4a82bd8f0893c7d00a87a36ed36fa}{Shape()}};
s.addPoint(a);
s.addPoint(b);
s.addPoint(new Point(1, 1));
for (\mbox{\hyperlink{classedu_1_1duke_1_1_point}{Point}} p : s.getPoints()) \{
    System.out.println(p);
\}
\end{DoxyPre}


A {\ttfamily \mbox{\hyperlink{classedu_1_1duke_1_1_shape}{Shape}}} can also be created using file in the following format\+:


\begin{DoxyPre}
3, 4
2, 5
1, 1
\end{DoxyPre}


Example file usage\+:


\begin{DoxyPre}
\mbox{\hyperlink{classedu_1_1duke_1_1_file_resource}{FileResource}} fr = new FileResource();
\mbox{\hyperlink{classedu_1_1duke_1_1_shape}{Shape}} s = new Shape(fr);
for (\mbox{\hyperlink{classedu_1_1duke_1_1_point}{Point}} p : s.getPoints()) \{
    System.out.println(p);
\}
\end{DoxyPre}
